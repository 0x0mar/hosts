####Usage
1. 把hosts文件覆盖C:\Windows\System32\drivers\etc
2. 如果失效，使用下面的Git命令更新,或者到[Github](https://gist.github.com/DingSoung/5f00ad4e7fa8006c8dde)下载

		git pull
