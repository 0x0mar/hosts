####Usage
1. [打开hosts](https://raw.githubusercontent.com/DingSoung/hosts/master/hosts)，全选复制,覆盖C:\Windows\System32\drivers\etc\hosts(windows), \etc\hosts (Android Mac Linux)并保存
2. 如果失效，到右边提交issues
