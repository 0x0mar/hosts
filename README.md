####Usage
1. 把hosts文件覆盖C:\Windows\System32\drivers\etc(windows), \etc (Android Mac Linux)
2. 如果失效，使用下面的Git命令更新,或者到[Github](https://github.com/DingSoung/hosts)下载

		git pull
